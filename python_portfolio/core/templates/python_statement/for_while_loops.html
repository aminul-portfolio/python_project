🔁 For & While Loops

Loops are fundamental to automating repetitive tasks. Below are real-world use cases using both for and while loops.

🛒 Loop through a shopping list
items = ['Milk', 'Eggs', 'Bread']
for item in items:
    print("Buy:", item)
  • 🔹 Function Used: for loop
  • 📘 Learned: Iterating over list items
  • ♻️ Reusability: Inventory or task automation
  • 💬 Comments: Simple list processing with loop
➗ Print even numbers in a range
for num in range(2, 11, 2):
    print(num)
  • 🔹 Function Used: range with step
  • 📘 Learned: Custom step iteration
  • ♻️ Reusability: Pattern generation, filtering
  • 💬 Comments: Efficient numeric filtering
➕ Sum values from a list
prices = [10, 15, 20]
total = 0
for price in prices:
    total += price
print("Total:", total)
  • 🔹 Function Used: Accumulator with for loop
  • 📘 Learned: Summing list elements
  • ♻️ Reusability: Financial calculators
  • 💬 Comments: Common in billing or checkout
🔤 Display characters in a name
name = "Sumon"
for char in name:
    print(char)
  • 🔹 Function Used: for loop on string
  • 📘 Learned: String iteration
  • ♻️ Reusability: Text processing, validation
  • 💬 Comments: Useful in string parsing tasks
🔢 Count down from 5
count = 5
while count > 0:
    print(count)
    count -= 1
  • 🔹 Function Used: while loop with decrement
  • 📘 Learned: Controlled countdown
  • ♻️ Reusability: Timers, retries, delays
  • 💬 Comments: Perfect for game logic or delays
🔐 Password retry attempt
password = ""
while password != "admin123":
    password = input("Enter password: ")
  • 🔹 Function Used: while with input check
  • 📘 Learned: Loop until condition met
  • ♻️ Reusability: Security & login forms
  • 💬 Comments: Mimics real-world authentication logic
⏳ Wait for condition to become true
ready = False
while not ready:
    user_input = input("Type 'yes' when ready: ")
    if user_input == 'yes':
        ready = True
  • 🔹 Function Used: while + conditional break
  • 📘 Learned: Wait loops
  • ♻️ Reusability: Readiness or polling check
  • 💬 Comments: Great for controlled flow
🔍 Validate user input
age = -1
while age < 0:
    age = int(input("Enter a valid age: "))
  • 🔹 Function Used: Input validation loop
  • 📘 Learned: Validation logic
  • ♻️ Reusability: User form processing
  • 💬 Comments: Helps sanitize user input
