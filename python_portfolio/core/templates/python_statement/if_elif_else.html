⚖️ Python Comparison Operators

Comparison operators are used to compare two values and return True or False.

1️⃣ Equal vs Not Equal
a = 5
b = 10
print(a == b)  # False
print(a != b)  # True
  • 🔹 Function Used: ==, !=
  • 📘 Learned: Equality and inequality comparison
  • ♻️ Reusability: Used in conditional checks
  • 💬 Comments: Essential in decision making
2️⃣ Greater Than / Less Than
age = 18
if age >= 18:
    print("You can vote.")
else:
    print("Too young to vote.")
  • 🔹 Function Used: >=, <=
  • 📘 Learned: Age validation logic
  • ♻️ Reusability: Threshold-based conditions
  • 💬 Comments: Common in form validations
3️⃣ Sorting with Comparisons
numbers = [3, 7, 2, 5]
if numbers[0] < numbers[-1]:
    print("First is smaller than last")
  • 🔹 Function Used: <
  • 📘 Learned: Comparing list elements
  • ♻️ Reusability: Helpful in sorting logic
  • 💬 Comments: Works on any comparable values
