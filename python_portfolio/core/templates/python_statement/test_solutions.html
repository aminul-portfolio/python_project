1️⃣ Compare two numbers
x = 10
y = 20
print(x < y)
  • 🔹 Function Used: <
  • 📘 Learned: Comparing values using less-than.
  • ♻️ Reusability: Useful in conditions and filters.
  • 💬 Comments: Returns True since 10 is less than 20.
2️⃣ Check for equality
a = 5
b = 5
print(a == b)
  • 🔹 Function Used: ==
  • 📘 Learned: Checking if two values are equal.
  • ♻️ Reusability: Fundamental for if-statements and comparisons.
  • 💬 Comments: Returns True since both values are equal.
3️⃣ Not equal condition
x = 7
y = 3
print(x != y)
  • 🔹 Function Used: !=
  • 📘 Learned: Checking if two values are not equal.
  • ♻️ Reusability: Commonly used in loops and validation.
  • 💬 Comments: Returns True since values differ.
4️⃣ Greater than or equal to
score = 85
print(score >= 50)
  • 🔹 Function Used: >=
  • 📘 Learned: Checks if a value meets or exceeds a threshold.
  • ♻️ Reusability: Used in grading systems or minimum criteria checks.
  • 💬 Comments: Returns True since 85 is greater than 50.
5️⃣ Compare strings
print("apple" < "banana")
  • 🔹 Function Used: <
  • 📘 Learned: Strings can be compared lexicographically.
  • ♻️ Reusability: Helpful in sorting and filtering text.
  • 💬 Comments: Returns True since 'a' comes before 'b'.
6️⃣ Identity check with is
a = [1, 2]
b = a
print(a is b)
  • 🔹 Function Used: is
  • 📘 Learned: Checks if two variables refer to the same object.
  • ♻️ Reusability: Useful for checking reference equality.
  • 💬 Comments: Returns True since both point to the same list.
7️⃣ Membership check with in
print(3 in [1, 2, 3, 4])
  • 🔹 Function Used: in
  • 📘 Learned: Checks if an item exists in a sequence.
  • ♻️ Reusability: Common for validation and filtering.
  • 💬 Comments: Returns True because 3 is in the list.
8️⃣ Chain comparison
x = 5
print(1 < x < 10)
  • 🔹 Function Used: chaining operators
  • 📘 Learned: Python supports chain comparison.
  • ♻️ Reusability: Concise way to check a range.
  • 💬 Comments: Returns True if all comparisons are True.
9️⃣ Less than or equal to
age = 18
print(age <= 18)
  • 🔹 Function Used: <=
  • 📘 Learned: Compares if value is less than or equal to another.
  • ♻️ Reusability: Ideal for age or limit checks.
  • 💬 Comments: Returns True because 18 is equal to 18.
🔟 Identity check with is not
a = [1, 2]
b = [1, 2]
print(a is not b)
  • 🔹 Function Used: is not
  • 📘 Learned: Checks if two variables do not refer to the same object.
  • ♻️ Reusability: Use to ensure objects are separate.
  • 💬 Comments: Returns True because a and b are different objects.
