📦 Python Modules & Packages

This section introduces Python modules and packages, essential tools for organizing code and reusing functionality across projects.

🔹 Example 1: Importing a Standard Module
import math

result = math.sqrt(16)
print("Square root:", result)

📘 What you learn: How to import and use built-in Python modules.

♻️ Reusability: Use built-in modules like math, random, datetime to simplify calculations and operations.

💬 Comments: math.sqrt() is a function from the math module for square root calculation.

🔹 Example 2: Creating and Importing a Custom Module
# File: my_utils.py
def greet(name):
    return f"Hello, {name}!"

# In your main script
import my_utils
print(my_utils.greet("Sumon"))

📘 What you learn: How to create a custom module and use it in other scripts.

♻️ Reusability: Write reusable utility functions in separate files.

💬 Comments: Use the same folder or PYTHONPATH to locate modules correctly.

🔹 Example 3: Using from-import syntax
from math import pi, ceil
print("Pi:", pi)
print("Ceil of 4.2:", ceil(4.2))

📘 What you learn: How to import specific functions or constants.

♻️ Reusability: Keeps namespace clean and improves readability.

💬 Comments: Only imports what's needed from the module.
