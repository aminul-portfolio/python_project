🛠️ Real-World Python Error Handling Examples

Explore common Python exceptions with real-life code and test your knowledge with a 10-question quiz.

1️⃣ ZeroDivisionError
try:
    result = 10 / 0
except ZeroDivisionError:
    print("You can't divide by zero!")
  • 🔹 Function Used: try-except block
  • 📘 Learned: Catching division by zero
  • ♻️ Reusability: Useful in math-based user input
  • 💬 Comments: Prevents app crash from divide-by-zero
2️⃣ ValueError
try:
    num = int("abc")
except ValueError:
    print("Invalid conversion from string to int")
  • 🔹 Function Used: int()
  • 📘 Learned: Handling bad type conversion
  • ♻️ Reusability: Common in form input validation
  • 💬 Comments: Guides users on expected input
3️⃣ FileNotFoundError
try:
    with open("file.txt") as f:
        content = f.read()
except FileNotFoundError:
    print("File not found!")
  • 🔹 Function Used: open()
  • 📘 Learned: How to catch missing files
  • ♻️ Reusability: File management apps
  • 💬 Comments: Avoids crashes due to missing files
4️⃣ KeyError
try:
    name = data["name"]
except KeyError:
    print("Key does not exist!")
  • 🔹 Function Used: Dictionary access
  • 📘 Learned: Detecting missing dictionary keys
  • ♻️ Reusability: Used in API data handling
  • 💬 Comments: Helpful for debugging missing keys
5️⃣ IndexError
try:
    value = my_list[10]
except IndexError:
    print("Index out of range!")
  • 🔹 Function Used: List indexing
  • 📘 Learned: Handling list boundaries
  • ♻️ Reusability: Data extraction in loops
  • 💬 Comments: Prevents app from crashing
6️⃣ ImportError
try:
    from my_module import something
except ImportError:
    print("Module or object not found!")
  • 🔹 Function Used: Import
  • 📘 Learned: Missing module handling
  • ♻️ Reusability: Safe dependency loading
  • 💬 Comments: Helps during deployment
7️⃣ AssertionError
try:
    assert 2 + 2 == 5
except AssertionError:
    print("Assertion failed!")
  • 🔹 Function Used: assert
  • 📘 Learned: Logical validation
  • ♻️ Reusability: Testing and debugging
  • 💬 Comments: Helps verify assumptions
8️⃣ AttributeError
try:
    x = None
    x.strip()
except AttributeError:
    print("Object has no attribute!")
  • 🔹 Function Used: Object method access
  • 📘 Learned: Safe method calls
  • ♻️ Reusability: Prevents NoneType bugs
  • 💬 Comments: Important in form handlers
9️⃣ ValueError
try:
    int('123.45')
except ValueError:
    print("Float string to int conversion error!")
  • 🔹 Function Used: int()
  • 📘 Learned: Parsing errors
  • ♻️ Reusability: File parsing, input handling
  • 💬 Comments: Avoid bad type conversion
🔟 Custom Exception
try:
    raise Exception("Custom error raised!")
except Exception as e:
    print(f"Error: {e}")
  • 🔹 Function Used: raise Exception()
  • 📘 Learned: Throwing your own error
  • ♻️ Reusability: Application-specific messages
  • 💬 Comments: Useful in business rules
