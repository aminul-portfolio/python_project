1️⃣ 📂 Load and Show Image
from PIL import Image
image = Image.open("example.jpg")
image.show()
  • 🔹 Function Used: Image.open(), image.show()
  • 📘 Learned: Open and view images using PIL.
  • ♻️ Reusability: Base for most image processing pipelines.
  • 💬 Comments: Simple image visualization with PIL.
2️⃣ 🎨 Apply Color Filter
r, g, b = image.split()
purple = Image.merge("RGB", (r, g, b))
  • 🔹 Function Used: split(), merge()
  • 📘 Learned: Work with RGB channels individually.
  • ♻️ Reusability: Great for building filters and effects.
  • 💬 Comments: Color channels manipulation.
3️⃣ 🎭 Masking Images
mask = Image.open("mask.png")
image.paste(purple, (0, 0), mask)
  • 🔹 Function Used: paste()
  • 📘 Learned: Overlay one image onto another with masking.
  • ♻️ Reusability: Used in layering and compositing images.
  • 💬 Comments: Transparency through masking.
4️⃣ ✏️ Annotate with Text
from PIL import ImageDraw
img = Image.open("pencils.jpg")
draw = ImageDraw.Draw(img)
draw.text((20, 20), "Hello!", fill="white")
  • 🔹 Function Used: ImageDraw.Draw(), draw.text()
  • 📘 Learned: Add text directly to images.
  • ♻️ Reusability: Create instructional diagrams or watermarks.
  • 💬 Comments: Adds text annotations easily.
