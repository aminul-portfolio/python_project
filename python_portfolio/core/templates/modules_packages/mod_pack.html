📦 Modules and Packages

Explore how Python modules and packages are used to organize and reuse code effectively.

1️⃣ mymodule.py
def my_func():
    print("Hey I am in mymodule.py")
  • 🔹 Function Used: print()
  • 📘 Learned: How to define and create a custom module.
  • ♻️ Reusability: You can reuse my_func() in other scripts.
  • 💬 Comments: Modular design encourages cleaner code organization.
2️⃣ myprogram.py
from mymodule import my_func

my_func()
  • 🔹 Function Used: import
  • 📘 Learned: How to use functions from another file/module.
  • ♻️ Reusability: Encourages importing instead of rewriting code.
  • 💬 Comments: Python’s import system is key to scaling code.
3️⃣ some_main_script.py
from mymainpackage import some_main_script
from mymainpackage.subpackage import mysubscript

some_main_script.report_main()
mysubscript.sub_report()
  • 🔹 Function Used: from ... import ...
  • 📘 Learned: How to import and use nested packages/modules.
  • ♻️ Reusability: Code becomes reusable and easier to maintain.
  • 💬 Comments: Useful for large projects like Django or Flask apps.
4️⃣ mysubscript.py
def sub_report():
    print("Hey I am a function inside mysubscript.py")
  • 🔹 Function Used: print()
  • 📘 Learned: Submodules provide specific, focused functionality.
  • ♻️ Reusability: Can be reused in parent scripts or unit tests.
  • 💬 Comments: Encourages modular programming practice.
