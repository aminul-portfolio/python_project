🧩 17 - Bonus: Widget Interactivity and Styling

Explore advanced usage of widgets in Python using ipywidgets for UI in Jupyter notebooks.

1️⃣ Interactive Math Operation
# Widgets to adjust parameters
from ipywidgets import interactive
def f(x, y):
    return x * y

interactive(f, x=10, y=5)
  • 🔹 Function Used: interactive()
  • 📘 Learned: Create dynamic user inputs with real-time function feedback
  • ♻️ Reusability: Parameter tuning for models or calculators
  • 💬 Comments: Ideal for math apps or simulations in notebooks
2️⃣ Dynamic Text Input
import ipywidgets as widgets
widgets.Text(value='Hello', placeholder='Type something', description='Text:')
  • 🔹 Function Used: widgets.Text()
  • 📘 Learned: Create basic user input fields for text data
  • ♻️ Reusability: Forms, dashboards, or chatbot UIs
  • 💬 Comments: Acts like a simple text box component
3️⃣ Custom Layout with VBox
from ipywidgets import Button, VBox
button1 = Button(description="Click 1")
button2 = Button(description="Click 2")
VBox([button1, button2])
  • 🔹 Function Used: VBox()
  • 📘 Learned: Stack UI components vertically
  • ♻️ Reusability: UI menus, dashboards
  • 💬 Comments: Useful for simple layout control
4️⃣ Event Handler for Buttons
def on_click(b):
    print("Button clicked!")

button = widgets.Button(description="Click Me")
button.on_click(on_click)
button
  • 🔹 Function Used: on_click()
  • 📘 Learned: Capture button press actions
  • ♻️ Reusability: User controls, submissions, games
  • 💬 Comments: Common in quiz apps or UI confirmations
