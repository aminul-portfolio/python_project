🧵 Python Strings

Strings in Python are sequences of characters enclosed in quotes. They support powerful operations and methods.

1️⃣ Email Username Extractor
email = "sumon@example.com"
username = email.split("@")[0]
print("Username:", username)  # Output: sumon
  • 🔹 Function Used: split()
  • 📘 Learned: Extract username from email
  • ♻️ Reusability: Login systems, parsing
  • 💬 Comments: Useful in email automation
2️⃣ Password Masking
password = "mysecret123"
masked = "*" * len(password)
print("Masked Password:", masked)  # Output: ***********
  • 🔹 Function Used: len(), *
  • 📘 Learned: Hiding password length
  • ♻️ Reusability: Login UI
  • 💬 Comments: Security-related display
3️⃣ Name Formatting
name = "sumon islam"
formatted = name.title()
print(formatted)  # Output: Sumon Islam
  • 🔹 Function Used: title()
  • 📘 Learned: Capitalizing each word
  • ♻️ Reusability: Clean names in forms
  • 💬 Comments: Ensures clean display
4️⃣ Word Counter
sentence = "Python is easy to learn"
word_count = len(sentence.split())
print("Words:", word_count)  # Output: 5
  • 🔹 Function Used: split(), len()
  • 📘 Learned: Count number of words
  • ♻️ Reusability: Analytics, text processing
  • 💬 Comments: Accurate for whitespace-split
