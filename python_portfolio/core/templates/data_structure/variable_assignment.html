📘 Python Variable Assignment

Variables are names that refer to values. Use the = operator to assign a value.

1️⃣ Product Inventory
product_name = "Laptop"
quantity = 5
price = 700.00
total_value = quantity * price
print(total_value)  # Output: 3500.0
  • 🔹 Function Used: Variable assignment, multiplication
  • 📘 Learned: Calculating total inventory value
  • ♻️ Reusability: Useful for retail, accounting apps
  • 💬 Comments: Combines quantity and price to derive totals
2️⃣ Temperature Monitoring
city = "London"
temperature_c = 19.5
feels_like = temperature_c - 2
print(f"Feels like {feels_like}°C in {city}")
  • 🔹 Function Used: Subtraction, string interpolation
  • 📘 Learned: Adjusting temperature readings
  • ♻️ Reusability: Weather apps, IoT sensor data
  • 💬 Comments: Shows dynamic string output with variables
3️⃣ Bank Account Tracker
account_holder = "Sumon"
balance = 1200
withdrawal = 300
remaining = balance - withdrawal
print(f"{account_holder} has £{remaining} left.")
  • 🔹 Function Used: Variable subtraction, string formatting
  • 📘 Learned: Deducting values and formatting user messages
  • ♻️ Reusability: Online banking, personal finance tracking
  • 💬 Comments: Ideal pattern for tracking account transactions
4️⃣ Exam Score Calculator
student = "Shyni"
math_score = 85
english_score = 78
average = (math_score + english_score) / 2
print(f"{student}'s average score is {average}")
  • 🔹 Function Used: Addition, division, variable grouping
  • 📘 Learned: Calculating averages and personalized output
  • ♻️ Reusability: Education apps, student performance dashboards
  • 💬 Comments: Demonstrates clean calculation and output format
