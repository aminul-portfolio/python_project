Python Comparison Operators

Comparison operators allow us to compare values and return Boolean results. These are essential in decision-making logic and conditions.

1️⃣ Age Check
age = 18
print(age >= 18)  # Output: True
  • 🔹 Function Used: >=
  • 📘 Learned: Check if someone is eligible by age
  • ♻️ Reusability: Age-based validation
  • 💬 Comments: Used in registration checks
2️⃣ Password Match
stored = "sumon123"
input_password = "Sumon123"
print(stored == input_password)  # Output: False (case-sensitive)
  • 🔹 Function Used: ==
  • 📘 Learned: Compare two strings
  • ♻️ Reusability: Login validation
  • 💬 Comments: Case-sensitive check
3️⃣ Salary Range (Chained Comparison)
salary = 3500
print(3000 <= salary <= 5000)  # Output: True
  • 🔹 Function Used: <= (chained)
  • 📘 Learned: Range validation
  • ♻️ Reusability: Salary grading
  • 💬 Comments: Clean syntax for range checks
4️⃣ Item Price Comparison
item1 = 120
item2 = 150
print(item1 < item2)  # Output: True
  • 🔹 Function Used: <
  • 📘 Learned: Price comparison
  • ♻️ Reusability: E-commerce, pricing logic
  • 💬 Comments: Useful for sorting or filtering
