🧾 Question-by-Question Solutions

  1. Q1: What is the result of 2 ** 3?
    ✅ Answer: 8
    📘 Explanation: ** is the exponent operator. 23 = 8.

  2. Q2: What will len("Hello World") return?
    ✅ Answer: 11
    📘 Explanation: It counts all characters including the space.

  3. Q3: What is the output of "hello"[1]?
    ✅ Answer: "e"
    📘 Explanation: Indexing starts at 0. "hello"[1] → "e".

  4. Q4: Which of these creates a list?
    ✅ Answer: [1,2,3]
    📘 Explanation: Square brackets define a list. Tuples use parentheses, sets use curly braces.

  5. Q5: Which method adds an item to a list?
    ✅ Answer: append()
    📘 Explanation: append() adds a single item to the end of a list.

  6. Q6: What will {'a':10, 'b':20}['b'] return?
    ✅ Answer: 20
    📘 Explanation: Access dictionary values using keys. Here, key 'b' → value 20.

  7. Q7: Which returns all keys from a dictionary?
    ✅ Answer: keys()
    📘 Explanation: .keys() returns all keys from a dictionary object.

  8. Q8: What is the result of set([1,1,2,3])?
    ✅ Answer: {1,2,3}
    📘 Explanation: Sets automatically remove duplicates.

  9. Q9: What is the output of bool(0)?
    ✅ Answer: False
    📘 Explanation: Zero is considered False in boolean context.

  10. Q10: Which is a valid boolean expression?
    ✅ Answer: All of the above
    📘 Explanation: 5 > 3, 'a' in 'cat', and True or False are all valid.
