📚 Python Dictionaries

Dictionaries store key-value pairs—perfect for structured data like user profiles, product info, and configurations.

1️⃣ User Profile
user = {
    "name": "Sumon",
    "email": "sumon@example.com",
    "age": 30
}
print(user["email"])  # Output: sumon@example.com
  • 🔹 Function Used: Dictionary access by key
  • 📘 Learned: Retrieve specific value
  • ♻️ Reusability: User systems
  • 💬 Comments: Simple lookup pattern
2️⃣ Product Catalog
product = {
    "id": 1001,
    "name": "Laptop",
    "price": 899.99
}
print(product["name"])  # Output: Laptop
  • 🔹 Function Used: Dictionary definition
  • 📘 Learned: Key-value data modeling
  • ♻️ Reusability: Product listings
  • 💬 Comments: Great for APIs
3️⃣ Inventory Management
inventory = {"apple": 10, "banana": 5}
inventory["banana"] += 2
print(inventory)  # Output: {'apple': 10, 'banana': 7}
  • 🔹 Function Used: Update values
  • 📘 Learned: Modify existing key
  • ♻️ Reusability: Stock tracking
  • 💬 Comments: Real-world inventory logic
4️⃣ Safe Access with get()
data = {"key1": "value1"}
print(data.get("key2", "Not Found"))  # Output: Not Found
  • 🔹 Function Used: get()
  • 📘 Learned: Prevent key errors
  • ♻️ Reusability: Defensive coding
  • 💬 Comments: Avoids crashes
