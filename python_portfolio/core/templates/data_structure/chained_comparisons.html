⛓️ Chained Comparison Operators

Chained comparisons in Python let you check multiple conditions in a single, readable line.

1️⃣ Age Between Range
age = 25
print(18 <= age <= 30)  # Output: True
  • 🔹 Function Used: <= (less than or equal)
  • 📘 Learned: Check if a value is in a range
  • ♻️ Reusability: Validating age limits
  • 💬 Comments: Clean readable comparisons
2️⃣ Score Range Validation
score = 82
print(60 <= score <= 100)  # Output: True
  • 🔹 Function Used: <= comparison
  • 📘 Learned: Check if score is valid
  • ♻️ Reusability: Exams, evaluations
  • 💬 Comments: Compact and clear logic
3️⃣ Temperature Comfort Check
temp = 21
print(20 <= temp <= 25)  # Output: True
  • 🔹 Function Used: <= operator
  • 📘 Learned: Determine comfortable range
  • ♻️ Reusability: Thermostat or climate control
  • 💬 Comments: Chain of range validation
4️⃣ Salary Tier
salary = 5000
print(3000 < salary < 7000)  # Output: True
  • 🔹 Function Used: < operator
  • 📘 Learned: Detect salary range
  • ♻️ Reusability: Job filtering, classification
  • 💬 Comments: Easy way to validate salary bands
