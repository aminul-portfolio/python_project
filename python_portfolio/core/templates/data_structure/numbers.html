📘 Python Variable Assignment

Variables are names that refer to values. Use the = operator to assign a value.

1️⃣ Budget Calculation
income = 3000
expenses = 2500
savings = income - expenses
print(savings)  # Output: 500
  • 🔹 Function Used: Basic arithmetic (-)
  • 📘 Learned: Subtracting to find savings
  • ♻️ Reusability: Budgeting, accounting
  • 💬 Comments: Simple numeric operation
2️⃣ Percentage Discount
price = 120
discount = 20
final_price = price * ((100 - discount) / 100)
print(final_price)  # Output: 96.0
  • 🔹 Function Used: Float division
  • 📘 Learned: Percentage-based reduction
  • ♻️ Reusability: Shopping, eCommerce
  • 💬 Comments: Watch decimals
3️⃣ Round and Absolute
print(round(7.6))   # Output: 8
print(abs(-4))      # Output: 4
  • 🔹 Function Used: round(), abs()
  • 📘 Learned: Rounding and positive conversion
  • ♻️ Reusability: Data cleaning, finance
  • 💬 Comments: abs() helps track deviation
