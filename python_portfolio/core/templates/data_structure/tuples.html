🐍 Python Tuples

Tuples are immutable, ordered collections used to store multiple items. They are useful when your data shouldn't change.

1️⃣ Coordinate System
location = (51.5074, 0.1278)
print("Latitude:", location[0])
print("Longitude:", location[1])
  • 🔹 Function Used: Indexing tuple
  • 📘️ Learned: How to store GPS coordinates
  • ♻️ Reusability: Mapping, navigation apps
  • 💬 Comments: Tuples maintain order
2️⃣ Return Multiple Values
def get_user():
    return ("Sumon", 30)

name, age = get_user()
print(name, age)
  • 🔹 Function Used: Tuple return and unpacking
  • 📘️ Learned: Functions can return multiple values
  • ♻️ Reusability: APIs, DB queries
  • 💬 Comments: Simplifies value grouping
3️⃣ Immutable Data
colors = ("red", "green", "blue")
# colors[0] = "yellow" ❌ TypeError
  • 🔹 Function Used: Tuple immutability
  • 📘️ Learned: Tuples cannot be changed
  • ♻️ Reusability: Fixed configuration
  • 💬 Comments: Great for constant data
4️⃣ Tuple Packing & Unpacking
data = ("Python", 3.12)
language, version = data
print(language, version)
  • 🔹 Function Used: Tuple unpacking
  • 📘️ Learned: Extract values from tuples
  • ♻️ Reusability: Return parsing
  • 💬 Comments: Makes code cleaner
