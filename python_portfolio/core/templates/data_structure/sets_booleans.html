🧩 Python Sets & Booleans

Sets are unordered collections of unique elements. Booleans represent truth values and are often used in conditionals and comparisons.

1️⃣ Removing Duplicates
emails = ["a@x.com", "b@x.com", "a@x.com"]
unique_emails = set(emails)
print(unique_emails)  # {'a@x.com', 'b@x.com'}
  • 🔹 Function Used: set()
  • 📘 Learned: Eliminate duplicates
  • ♻️ Reusability: Email deduplication, data cleanup
  • 💬 Comments: Fast and easy unique filtering
2️⃣ Membership Test
items = {"apple", "banana", "orange"}
print("banana" in items)  # True
  • 🔹 Function Used: in
  • 📘 Learned: Membership testing
  • ♻️ Reusability: Access control, validation checks
  • 💬 Comments: Efficient lookup in sets
3️⃣ Boolean Logic in Access Control
logged_in = True
is_admin = False
if logged_in and is_admin:
    print("Access granted")
else:
    print("Access denied")
  • 🔹 Function Used: and, if-else
  • 📘 Learned: Boolean logic in flow control
  • ♻️ Reusability: Login systems, permissions
  • 💬 Comments: Clean boolean decision flow
4️⃣ Set Operations
a = {1, 2, 3}
b = {3, 4, 5}
print(a & b)  # {3}
print(a | b)  # {1, 2, 3, 4, 5}
  • 🔹 Function Used: & (intersection), | (union)
  • 📘 Learned: Combining and comparing sets
  • ♻️ Reusability: Analytics, recommendation systems
  • 💬 Comments: Elegant math-like set ops
