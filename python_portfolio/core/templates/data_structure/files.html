📁 Python File Handling

File handling in Python is essential for reading and writing data from external sources like text files, logs, or user inputs.

1️⃣ Reading a Text File
with open("data.txt", "r") as file:
    content = file.read()
print(content)
  • 🔹 Function Used: open(), read()
  • 📘 Learned: How to read file content
  • ♻️ Reusability: Log readers, data importers
  • 💬 Comments: File must exist or error occurs
2️⃣ Writing to a File
with open("output.txt", "w") as file:
    file.write("Hello, Sumon!")
  • 🔹 Function Used: open(), write()
  • 📘 Learned: How to write data to files
  • ♻️ Reusability: Report generation
  • 💬 Comments: Overwrites existing files
3️⃣ Appending to a File
with open("log.txt", "a") as file:
    file.write("\nNew log entry.")
  • 🔹 Function Used: open() with 'a'
  • 📘 Learned: Add content to existing file
  • ♻️ Reusability: Logging, tracking
  • 💬 Comments: Doesn’t remove existing data
4️⃣ Checking if File Exists
import os
if os.path.exists("data.txt"):
    print("File found")
else:
    print("File not found")
  • 🔹 Function Used: os.path.exists()
  • 📘 Learned: Validate file presence
  • ♻️ Reusability: Error handling, safe operations
  • 💬 Comments: Requires importing os module
