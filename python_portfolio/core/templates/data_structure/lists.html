🧾 Python Lists

Lists are ordered, mutable collections in Python used to store multiple items.

1⃣ Shopping Cart
cart = ["milk", "bread", "eggs"]
cart.append("butter")
print(cart)  # ['milk', 'bread', 'eggs', 'butter']
  • 🔹 Function Used: append()
  • 📘 Learned: Add items dynamically to a list
  • ♻️ Reusability: eCommerce shopping logic
  • 💬 Comments: Great for dynamic data addition
2⃣ Remove Out-of-Stock Item
cart.remove("eggs")
print(cart)  # ['milk', 'bread', 'butter']
  • 🔹 Function Used: remove()
  • 📘 Learned: Remove specific values
  • ♻️ Reusability: Inventory adjustments
  • 💬 Comments: Removes first matching element
3⃣ Sorting Prices
prices = [12.5, 5.0, 9.99]
prices.sort()
print(prices)  # [5.0, 9.99, 12.5]
  • 🔹 Function Used: sort()
  • 📘 Learned: Sort numerical data
  • ♻️ Reusability: Price filtering, data prep
  • 💬 Comments: Sorts in-place
4⃣ Reversing Order
names = ["Sumon", "Rahim", "Karim"]
names.reverse()
print(names)  # ['Karim', 'Rahim', 'Sumon']
  • 🔹 Function Used: reverse()
  • 📘 Learned: Reverse list contents
  • ♻️ Reusability: UI logic, history display
  • 💬 Comments: Modifies original list
