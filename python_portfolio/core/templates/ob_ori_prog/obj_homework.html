This page combines real-world Object-Oriented Programming practice with solutions and a quiz to test your knowledge. It’s ideal for portfolio building and interview prep.

1️⃣ Create a Dog class
class Dog:
    def __init__(self, breed, name, spots):
        self.breed = breed
        self.name = name
        self.spots = spots
  • 🔹 Function Used: Class Constructor (__init__)
  • 📘 Learned: How to create a class and initialize attributes
  • ♻️ Reusability: Can reuse Dog class for any animal object creation
  • 💬 Comments: Constructor sets initial values when an object is created
2️⃣ Add class attribute
class Dog:
    species = 'mammal'
    def __init__(self, name):
        self.name = name
  • 🔹 Function Used: Class Attribute
  • 📘 Learned: How to define variables common to all instances
  • ♻️ Reusability: Use species across all Dog instances
  • 💬 Comments: Shared attribute declared outside any method
3️⃣ Add bark method
class Dog:
    def __init__(self, name):
        self.name = name
    def bark(self):
        print(f"Woof! My name is {self.name}")
  • 🔹 Function Used: Instance Method
  • 📘 Learned: How to define and use methods in a class
  • ♻️ Reusability: Add any behavior like bark or walk to the class
  • 💬 Comments: Method uses instance variable for output
4️⃣ Circle class
class Circle:
    pi = 3.14
    def __init__(self, radius=1):
        self.radius = radius
    def get_circumference(self):
        return 2 * Circle.pi * self.radius
  • 🔹 Function Used: Static Attribute and Method
  • 📘 Learned: Static value (pi) used with method
  • ♻️ Reusability: Flexible for any circle calculations
  • 💬 Comments: pi accessed via class name
5️⃣ Inheritance Example
class Animal:
    def __init__(self):
        print("Animal created")
class Dog(Animal):
    def __init__(self):
        super().__init__()
        print("Dog created")
  • 🔹 Function Used: Inheritance and super()
  • 📘 Learned: How child classes can inherit behavior from parent
  • ♻️ Reusability: Share base functionality across subclasses
  • 💬 Comments: super() calls parent constructor
6️⃣ Polymorphism Example
class Dog:
    def speak(self):
        return "Woof"
class Cat:
    def speak(self):
        return "Meow"
def pet_sound(pet):
    print(pet.speak())
  • 🔹 Function Used: Polymorphism
  • 📘 Learned: Different classes can share method names but behave differently
  • ♻️ Reusability: Common interface for different types
  • 💬 Comments: Enables flexible code
7️⃣ Abstract Base Class
from abc import ABC, abstractmethod
class Animal(ABC):
    @abstractmethod
    def speak(self):
        pass
  • 🔹 Function Used: Abstract Base Class
  • 📘 Learned: Force child classes to implement methods
  • ♻️ Reusability: Enforce consistent interface
  • 💬 Comments: Abstract classes cannot be instantiated
8️⃣ Special Methods
class Book:
    def __init__(self, title, author):
        self.title = title
        self.author = author
    def __str__(self):
        return f"{self.title} by {self.author}"
  • 🔹 Function Used: Special Method (__str__)
  • 📘 Learned: Customize behavior with special methods
  • ♻️ Reusability: Readable object representation
  • 💬 Comments: __str__ controls how object is printed
9️⃣ Encapsulation
class Person:
    def __init__(self, name):
        self.__name = name
    def get_name(self):
        return self.__name
  • 🔹 Function Used: Encapsulation (Private Variables)
  • 📘 Learned: Protect access to data
  • ♻️ Reusability: Hide internal logic
  • 💬 Comments: Double underscore makes attribute private
🔟 Composition
class Engine:
    def start(self):
        print("Engine starts")
class Car:
    def __init__(self):
        self.engine = Engine()
    def start(self):
        self.engine.start()
  • 🔹 Function Used: Composition
  • 📘 Learned: Combine objects to build functionality
  • ♻️ Reusability: Modular design with reusable components
  • 💬 Comments: Object inside another object
