📘 Advanced Python Objects & Context Managers

Learn powerful Python techniques with real-world examples of data structures, dictionary manipulation, and context managers.

1️⃣ Square Even Numbers
def square_evens(nums):
    return [x**2 for x in nums if x % 2 == 0]

print(square_evens([1, 2, 3, 4]))  # ➞ [4, 16]
  • 🔹 Function Used: list comprehension, modulus
  • 📘 Learned: Filtering and transforming collections in one line
  • ♻️ Reusability: Data cleaning, math filtering, batch processing
  • 💬 Comments: Squares even numbers from a list using one-liner logic
2️⃣ Word Frequency Counter
def word_count(text):
    result = {}
    for word in text.split():
        result[word] = result.get(word, 0) + 1
    return result

print(word_count("hello world hello"))  # ➞ {'hello': 2, 'world': 1}
  • 🔹 Function Used: split(), get()
  • 📘 Learned: Dictionary construction and safe updates
  • ♻️ Reusability: Natural language processing, logging, frequency analysis
  • 💬 Comments: Efficiently builds a word frequency dictionary
3️⃣ Reverse Dictionary
def reverse_dict(d):
    return {v: k for k, v in d.items()}

print(reverse_dict({'a': 1, 'b': 2}))  # ➞ {1: 'a', 2: 'b'}
  • 🔹 Function Used: dict comprehension, items()
  • 📘 Learned: Reversing key/value pairs elegantly
  • ♻️ Reusability: Data mapping, reverse lookup utilities
  • 💬 Comments: Inverts keys and values in a dictionary
4️⃣ Zip with List Comprehension
names = ['Alice', 'Bob']
scores = [85, 90]
result = [f"{n} scored {s}" for n, s in zip(names, scores)]
print(result)  # ➞ ['Alice scored 85', 'Bob scored 90']
  • 🔹 Function Used: zip(), f-string, list comprehension
  • 📘 Learned: Combining lists into formatted output
  • ♻️ Reusability: Report generation, summaries, UI formatting
  • 💬 Comments: Combines name-score pairs into readable text
5️⃣ With Statement & Context Manager
with open('sample.txt', 'r') as file:
    content = file.read()
    print(content)
class OpenUpper:
    def __init__(self, filename):
        self.file = open(filename, 'r')
    def __enter__(self):
        return self.file
    def __exit__(self, exc_type, exc_val, exc_tb):
        self.file.close()

with OpenUpper('sample.txt') as f:
    print(f.read())
  • 🔹 Function Used: with, __enter__, __exit__
  • 📘 Learned: Safe file handling with context managers
  • ♻️ Reusability: File operations, database connections, logging
  • 💬 Comments: Automates cleanup using built-in and custom context logic
