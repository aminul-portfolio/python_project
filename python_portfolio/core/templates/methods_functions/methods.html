🔧 String & List Methods in Python

Explore built-in methods for manipulating strings and lists in real-world Python programs.

1️⃣ upper()
name = "sumon"
print(name.upper())  # Output: SUMON
  • 🔹 Function Used: upper()
  • 📘 Learned: Convert string to uppercase
  • ♻️ Reusability: Formatting text
  • 💬 Comments: Useful for consistency
2️⃣ lower()
name = "SUMON"
print(name.lower())  # Output: sumon
  • 🔹 Function Used: lower()
  • 📘 Learned: Convert string to lowercase
  • ♻️ Reusability: Normalize data
  • 💬 Comments: Often used in search systems
3️⃣ split()
sentence = "I love Python"
print(sentence.split())  # Output: ['I', 'love', 'Python']
  • 🔹 Function Used: split()
  • 📘 Learned: Break string into list
  • ♻️ Reusability: Data parsing
  • 💬 Comments: Common in text processing
4️⃣ replace()
text = "Hello World"
print(text.replace("World", "Sumon"))  # Output: Hello Sumon
  • 🔹 Function Used: replace()
  • 📘 Learned: Replace parts of a string
  • ♻️ Reusability: Clean/modify text
  • 💬 Comments: Often used in templates
5️⃣ append()
fruits = ['apple']
fruits.append('banana')
print(fruits)  # Output: ['apple', 'banana']
  • 🔹 Function Used: append()
  • 📘 Learned: Add item to end of list
  • ♻️ Reusability: Dynamic lists
  • 💬 Comments: Key for growing data
6️⃣ pop()
fruits = ['apple', 'banana']
fruits.pop()
print(fruits)  # Output: ['apple']
  • 🔹 Function Used: pop()
  • 📘 Learned: Remove last item
  • ♻️ Reusability: Undo, stack logic
  • 💬 Comments: Returns popped value
7️⃣ sort()
nums = [3, 1, 2]
nums.sort()
print(nums)  # Output: [1, 2, 3]
  • 🔹 Function Used: sort()
  • 📘 Learned: Sort list in place
  • ♻️ Reusability: Ranking, leaderboard
  • 💬 Comments: Changes original list
8️⃣ reverse()
nums = [1, 2, 3]
nums.reverse()
print(nums)  # Output: [3, 2, 1]
  • 🔹 Function Used: reverse()
  • 📘 Learned: Reverse list items
  • ♻️ Reusability: Timeline flip, stack
  • 💬 Comments: In-place reversal
