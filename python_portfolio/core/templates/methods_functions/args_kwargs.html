üìò Real-World Examples with *args

1Ô∏è‚É£ Sum All Numbers

    return sum(args)

2Ô∏è‚É£ Multiply All Numbers
def multiply_all(*args):
    result = 1
    for num in args:
        result *= num
    return result

print(multiply_all(2, 3, 4))  # Output: 24
3Ô∏è‚É£ Display All Arguments
def show_args(*args):
    for i in args:
        print(i)

4Ô∏è‚É£ Combine with Required Arguments

    for name in names:
        print(f"{greeting}, {name}!")

greet("Hello", "Sumon", "Joy", "Tamim")

üìô Real-World Examples with **kwargs

5Ô∏è‚É£ Display Keyword Arguments
def print_details(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_details(name="Sumon", age=30)
6Ô∏è‚É£ Build a User Profile
def build_profile(**kwargs):
    return kwargs

print(profile)
7Ô∏è‚É£ Combine *args and **kwargs
def full_demo(*args, **kwargs):
    print("Args:", args)
    print("Kwargs:", kwargs)

full_demo(1, 2, 3, name="Sumon", age=30)
8Ô∏è‚É£ Pass **kwargs to Format Strings
def greeting(**kwargs):
    print("Hello, {name}! You are a {title}.".format(**kwargs))
