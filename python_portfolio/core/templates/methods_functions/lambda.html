⚡ Lambda Functions, Map & Filter

Lambda expressions are anonymous functions that simplify one-line function definitions. `map()` and `filter()` are functional tools that apply logic across iterables.

🧪 8 Real-World Examples

1️⃣ Square a Number
square = lambda x: x ** 2
print(square(5))
  • 🔹 Function Used: lambda
  • 📘 Learned: Define quick anonymous functions.
  • ♻️ Reusability: Useful for simple one-liner logic.
  • 💬 Comments: Squares the input, returns 25 for 5.
2️⃣ Add Two Numbers
add = lambda x, y: x + y
print(add(3, 4))
  • 🔹 Function Used: lambda
  • 📘 Learned: Add two values using a lambda.
  • ♻️ Reusability: Great for inline addition logic.
  • 💬 Comments: Returns 7 when passed 3 and 4.
3️⃣ Check if Even
is_even = lambda x: x % 2 == 0
print(is_even(6))
  • 🔹 Function Used: lambda, %
  • 📘 Learned: Use lambda for condition checks.
  • ♻️ Reusability: Handy for filtering data.
  • 💬 Comments: Returns True if number is even.
4️⃣ Extract First Character
first_char = lambda s: s[0]
print(first_char("Sumon"))
  • 🔹 Function Used: lambda, indexing
  • 📘 Learned: Access characters using lambda.
  • ♻️ Reusability: Simplifies string tasks.
  • 💬 Comments: Returns first character from string.
5️⃣ map(): Square List Items
nums = [1, 2, 3, 4]
squared = list(map(lambda x: x**2, nums))
print(squared)
  • 🔹 Function Used: map, lambda
  • 📘 Learned: Apply function to each item.
  • ♻️ Reusability: Great for bulk data transforms.
  • 💬 Comments: Returns list of squared numbers.
6️⃣ map(): Uppercase Names
names = ['sumon', 'joy', 'tamim']
uppercased = list(map(lambda x: x.upper(), names))
print(uppercased)
  • 🔹 Function Used: map, lambda, upper()
  • 📘 Learned: Change all strings to uppercase.
  • ♻️ Reusability: Perfect for formatting tasks.
  • 💬 Comments: Transforms each name to uppercase.
7️⃣ filter(): Keep Even Numbers
nums = [1, 2, 3, 4, 5]
evens = list(filter(lambda x: x % 2 == 0, nums))
print(evens)
  • 🔹 Function Used: filter, lambda
  • 📘 Learned: Filter elements based on condition.
  • ♻️ Reusability: Good for cleaning datasets.
  • 💬 Comments: Keeps only even numbers.
8️⃣ filter(): Filter Short Words
words = ['Hi', 'Welcome', 'To', 'Python']
short_words = list(filter(lambda w: len(w) <= 3, words))
print(short_words)
  • 🔹 Function Used: filter, lambda, len()
  • 📘 Learned: Keep only short words.
  • ♻️ Reusability: Useful for search/filter logic.
  • 💬 Comments: Returns words with 3 or fewer letters.
