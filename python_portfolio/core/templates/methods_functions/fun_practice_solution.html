💡 10 Real-World Function Examples

1️⃣ Greet a User
def greet(name):
    return f"Hello, {name}!"
  • 🔹 Function Used: greet(name)
  • 📘 Learned: How to define a function that returns a greeting message.
  • ♻️ Reusability: Can be used in any user-interaction system.
  • 💬 Comments: Returns a formatted string with the user's name.
2️⃣ Add Two Numbers
def add(a, b):
    return a + b
  • 🔹 Function Used: add(a, b)
  • 📘 Learned: Using a function to sum two values.
  • ♻️ Reusability: Common in calculators and numeric applications.
  • 💬 Comments: Returns the result of a + b.
3️⃣ Calculate Area of a Circle
def area_circle(radius):
    pi = 3.1416
    return pi * radius**2
  • 🔹 Function Used: area_circle(radius)
  • 📘 Learned: Applying math operations in functions.
  • ♻️ Reusability: Useful in geometry, physics, and simulations.
  • 💬 Comments: Multiplies π with the square of the radius.
4️⃣ Check Even or Odd
def is_even(num):
    return num % 2 == 0
  • 🔹 Function Used: is_even(num)
  • 📘 Learned: Using modulus operator to check for even numbers.
  • ♻️ Reusability: Helpful in filters and loops.
  • 💬 Comments: Returns True if number is divisible by 2.
5️⃣ Count Vowels in a String
def count_vowels(text):
    return sum(1 for char in text.lower() if char in 'aeiou')
  • 🔹 Function Used: count_vowels(text)
  • 📘 Learned: Iterate through strings and check character properties.
  • ♻️ Reusability: Can be used in text analysis or NLP preprocessing.
  • 💬 Comments: Converts text to lowercase and counts vowels.
6️⃣ Convert Celsius to Fahrenheit
def to_fahrenheit(c):
    return (c * 9/5) + 32
  • 🔹 Function Used: to_fahrenheit(c)
  • 📘 Learned: Applying temperature conversion logic in a function.
  • ♻️ Reusability: Great for weather apps and scientific calculations.
  • 💬 Comments: Uses formula (c * 9/5) + 32 to convert.

8️⃣ Reverse a String
def reverse_string(s):
    return s[::-1]
  • 🔹 Function Used: reverse_string(s)
  • 📘 Learned: Reverse strings using slicing.
  • ♻️ Reusability: Useful in palindromes, encoders, and data transformation.
  • 💬 Comments: s[::-1] reverses the characters of the string.
9️⃣ Check Prime
def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n**0.5)+1):
        if n % i == 0:
            return False
    return True
  • 🔹 Function Used: is_prime(n)
  • 📘 Learned: How to check for prime numbers using loops and conditions.
  • ♻️ Reusability: Fundamental in math-related programming.
  • 💬 Comments: Returns False if divisible by any number other than 1 and itself.
