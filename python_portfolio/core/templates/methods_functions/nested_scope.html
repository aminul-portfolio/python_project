🌀 Nested Functions and Variable Scope

Understanding how Python handles variable scope using nested functions and the LEGB rule (Local, Enclosing, Global, Built-in).

1️⃣ Basic Nested Function
def outer():
    print("Outer function")

    def inner():
        print("Inner function")

    inner()

outer()
  • 🔹 Function Used: Nested Function
  • 📘 Learned: How inner functions can be defined and called inside outer functions.
  • ♻️ Reusability: Common in closures and decorators.
  • 💬 Comments: Demonstrates that the inner function is scoped inside the outer function.
2️⃣ Return from Inner Function
def greet(name):
    def format_name(n):
        return n.capitalize()
    return f"Hello, {format_name(name)}"

print(greet("sumon"))
  • 🔹 Function Used: Inner function returning processed value
  • 📘 Learned: Using inner functions for encapsulating small tasks.
  • ♻️ Reusability: Promotes clean code and logic encapsulation.
  • 💬 Comments: Inner function is used to transform the input.
3️⃣ Inner Using Outer Variable
def multiply_by(factor):
    def inner(number):
        return number * factor
    return inner

double = multiply_by(2)
print(double(10))
  • 🔹 Function Used: Closure
  • 📘 Learned: Inner function accessing enclosing scope variable.
  • ♻️ Reusability: Excellent for creating customized logic like multipliers or filters.
  • 💬 Comments: This is an example of a closure in Python.
4️⃣ Global vs Local
x = "global"

def test():
    x = "local"
    print(x)

test()
print(x)
  • 🔹 Function Used: Variable scope assignment
  • 📘 Learned: Distinction between local and global variables.
  • ♻️ Reusability: Important in large applications to avoid scope bugs.
  • 💬 Comments: Shows how local scope does not affect global scope.
5️⃣ Enclosing Scope Example
def outer():
    msg = "enclosed"
    def inner():
        print(msg)
    inner()

outer()
  • 🔹 Function Used: Nested function with enclosing variable
  • 📘 Learned: LEGB rule in action with enclosing variable access.
  • ♻️ Reusability: Useful for data hiding and modular functions.
  • 💬 Comments: Demonstrates 'E' in LEGB—Enclosing scope.
6️⃣ Use global Keyword
x = "global"

def modify():
    global x
    x = "modified"

modify()
print(x)
  • 🔹 Function Used: global keyword
  • 📘 Learned: How to modify global variables from inside functions.
  • ♻️ Reusability: Useful in scripting where global state must be updated.
  • 💬 Comments: Avoid overuse; can lead to hard-to-track bugs.
7️⃣ Built-in Shadowing
str = "Not a function"
print(str(123))
  • 🔹 Function Used: Variable overriding built-in name
  • 📘 Learned: Risks of shadowing built-in functions.
  • ♻️ Reusability: Avoid such practice to prevent runtime errors.
  • 💬 Comments: Highlights importance of not naming variables after built-ins.
