This warm-up project demonstrates foundational Python skills using practical problem-solving examples. Ideal for interviews, assessments, and sharpening logical thinking.

1️⃣ Lesser of Two Evens
def lesser_of_two_evens(a, b):
    if a % 2 == 0 and b % 2 == 0:
        return min(a, b)
    else:
        return max(a, b)
  • 🔹 Function Used: `min()`, `max()`, modulo
  • 📘 Learned: Conditional logic to compare even numbers
  • ♻️ Reusability: Used in decision-making for competitive inputs
  • 💬 Comments: Handles both even/even and mixed odd/even scenarios
2️⃣ Animal Crackers
def animal_crackers(text):
    word_list = text.lower().split()
    return word_list[0][0] == word_list[1][0]
  • 🔹 Function Used: `split()`, `lower()`
  • 📘 Learned: How to compare first letters of two words
  • ♻️ Reusability: Great for text pattern checks
  • 💬 Comments: Assumes input has exactly two words
3️⃣ Makes Twenty
def makes_twenty(n1, n2):
    return (n1 + n2) == 20 or n1 == 20 or n2 == 20
  • 🔹 Function Used: arithmetic, boolean logic
  • 📘 Learned: Compound condition evaluation
  • ♻️ Reusability: Useful for threshold-based validations
  • 💬 Comments: Covers both sum and individual value comparisons
4️⃣ Old MacDonald
def old_macdonald(name):
    if len(name) > 3:
        return name[:1].capitalize() + name[1:3] + name[3:4].capitalize() + name[4:]
    return name.capitalize()
  • 🔹 Function Used: string slicing, `capitalize()`
  • 📘 Learned: Capitalization of specific characters
  • ♻️ Reusability: Can apply to name formatting systems
  • 💬 Comments: Modifies first and fourth letters conditionally
5️⃣ Master Yoda
def master_yoda(text):
    return ' '.join(text.split()[::-1])
  • 🔹 Function Used: `split()`, slicing
  • 📘 Learned: Reversing word order in a string
  • ♻️ Reusability: Great for string manipulation tasks
  • 💬 Comments: Used in natural language processing intros
