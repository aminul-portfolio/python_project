Debugger, Regex, Timing, and File Compression

Explore pdb debugging, regular expressions, code timing, and file compression using zip utilities.

1️⃣ pdb Debugging
x = [1,3,4]
y = 2
z = 3
result = y + z
print(result)
  • 🔹 Function Used: Variable assignment, print()
  • 📘 Learned: Understand simple operations before debugging.
  • ♻️ Reusability: Apply this pattern to debug any simple logic error.
  • 💬 Comments: Basic variable addition before tracing.
import pdb
x = [1,3,4]
y = 2
z = 3
result = y + z
pdb.set_trace()
result2 = y + x
  • 🔹 Function Used: pdb.set_trace()
  • 📘 Learned: Use breakpoints to debug runtime errors.
  • ♻️ Reusability: Helpful in larger applications for live debugging.
  • 💬 Comments: Debugging incorrect data types or logic flow.
2️⃣ Regular Expressions
text = "The person's phone number is 408-555-1234."
'phone' in text
  • 🔹 Function Used: in keyword
  • 📘 Learned: String containment test.
  • ♻️ Reusability: Basic filtering or keyword search.
  • 💬 Comments: Simple text search before regex.
import re
pattern = 'phone'
match = re.search(pattern, text)
  • 🔹 Function Used: re.search()
  • 📘 Learned: Start using pattern matching in Python.
  • ♻️ Reusability: Extendable for complex patterns and validation.
  • 💬 Comments: Returns match object if found.
3️⃣ Timing Code
def func_one(n):
    return [str(num) for num in range(n)]
  • 🔹 Function Used: list comprehension
  • 📘 Learned: Efficient loop-based string conversion.
  • ♻️ Reusability: Common in data processing tasks.
  • 💬 Comments: Clearer list-based syntax.
def func_two(n):
    return list(map(str, range(n)))
  • 🔹 Function Used: map()
  • 📘 Learned: Functional approach to transformation.
  • ♻️ Reusability: Useful in functional pipelines.
  • 💬 Comments: Works well with lambda too.
4️⃣ File Compression
f = open("new_file.txt", 'w+')
f.write("Here is some text")
f.close()
  • 🔹 Function Used: open(), write(), close()
  • 📘 Learned: Basic file I/O
  • ♻️ Reusability: Standard for creating or logging data files.
  • 💬 Comments: Always close files after writing.
import zipfile
comp_file = zipfile.ZipFile('comp_file.zip', 'w')
  • 🔹 Function Used: zipfile.ZipFile
  • 📘 Learned: Use Python to compress files.
  • ♻️ Reusability: Automate file packaging for backup or transfer.
  • 💬 Comments: Use 'with' for cleaner handling.
