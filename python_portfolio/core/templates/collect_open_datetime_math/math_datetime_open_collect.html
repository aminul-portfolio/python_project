📂 Python Utility Modules

Explore powerful built-in modules: collections, datetime, math, random, and open() for file handling

1️⃣ collections.Counter()
from collections import Counter

my_list = ['a', 'b', 'a', 'c', 'b', 'a']
print(Counter(my_list))
  • 🔹 Function Used: Counter()
  • 📘 What I Learn: Count occurrences in a list
  • ♻️ Reusability Insight: Frequency analysis, polls, logs
  • 💬 Comments: Count how often each element appears in a list.
2️⃣ collections.defaultdict()
from collections import defaultdict

counts = defaultdict(int)
counts['apple'] += 1
print(counts)
  • 🔹 Function Used: defaultdict()
  • 📘 What I Learn: Prevent KeyErrors with default values
  • ♻️ Reusability Insight: Grouping, dictionary building
  • 💬 Comments: Automatically initializes missing keys.
3️⃣ collections.namedtuple()
from collections import namedtuple

Point = namedtuple('Point', 'x y')
p1 = Point(10, 20)
print(p1.x, p1.y)
  • 🔹 Function Used: namedtuple()
  • 📘 What I Learn: Struct-like tuples with named fields
  • ♻️ Reusability Insight: Clean representations of objects
  • 💬 Comments: Provides field access to tuple elements by name.
4️⃣ collections.deque()
from collections import deque

d = deque()
d.append('first')
d.appendleft('zero')
print(d)
  • 🔹 Function Used: deque()
  • 📘 What I Learn: Fast appends/pops from both ends
  • ♻️ Reusability Insight: Task queues, undo systems
  • 💬 Comments: Allows quick append/appendleft operations.
5️⃣ open() file handling
with open('example.txt', 'w') as f:
    f.write("Hello, world!")
  • 🔹 Function Used: open()
  • 📘 What I Learn: Write text to a file safely
  • ♻️ Reusability Insight: Logging, data export, configs
  • 💬 Comments: Use 'with' to automatically close files.
6️⃣ datetime.now()
from datetime import datetime
print(datetime.now())
  • 🔹 Function Used: datetime.now()
  • 📘 What I Learn: Get current date and time
  • ♻️ Reusability Insight: Logging, timestamps, records
  • 💬 Comments: Retrieves real-time timestamp.
7️⃣ math.sqrt()
import math
print(math.sqrt(25))
  • 🔹 Function Used: math.sqrt()
  • 📘 What I Learn: Compute square roots
  • ♻️ Reusability Insight: Statistics, geometry, finance
  • 💬 Comments: Use math functions to handle numeric tasks.
8️⃣ random.choice()
import random
print(random.choice(['apple', 'banana', 'cherry']))
  • 🔹 Function Used: random.choice()
  • 📘 What I Learn: Select a random element from list
  • ♻️ Reusability Insight: Games, simulations, randomness
  • 💬 Comments: Great for drawing samples randomly.
